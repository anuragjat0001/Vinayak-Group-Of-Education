IMAGES FOLDER — Vinayak Group of Institutions Website

This folder is reserved for locally hosted image assets.

Current setup: All images are loaded from Unsplash CDN (free, no attribution required for usage).
When deploying with real campus photos:
1. Place images here in webp or jpg format
2. Recommended naming: campus-main.jpg, lab-chemistry.jpg, moot-court.jpg, etc.
3. Update src attributes in HTML to: assets/images/filename.jpg
4. Recommended sizes: Hero = 1920×1080, Cards = 800×600, Thumbnails = 400×300
